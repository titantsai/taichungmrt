# 台中捷運導覽PWA

## 功能

+ 捷運與台鐵轉乘路線圖

+ 旅程規劃
  - 查詢行車時間與票價 

+ 車站資訊
  * 站點資訊
    - 首末班車
    - 出口位置設施
    - 站內設施
  * 轉乘資訊
    - 轉乘停車場
    - Youbike 2.0即時數量
    - 公車路線列表(無查詢功能)
