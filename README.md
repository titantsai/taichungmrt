# 台中捷運導覽PWA

## 功能

+ 捷運與台鐵轉乘路線圖

+ 車站資訊
  * 站點資訊
    - 首末班車
    - 出口位置設施
    - 站內設施
  * 轉乘資訊
    - Youbike 2.0 即時動態
